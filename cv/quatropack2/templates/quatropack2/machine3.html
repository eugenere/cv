Super Seal™ Touch induction cap sealer
The industry's first interactive touch screen induction sealer


This single coil induction cap sealer was created to offer users increased productivity, an easier set-up and more control. The easy to use touch screen is available in multiple languages and comes with supervisory password protection and recipe management as standard features.

Touch screen control

  • Easy set-up and operation
  • Pre-programmed and fully integrated touch screen control for inspection and rejection
  • Remote start/stop and remote power control (auto start function available)
  • Password protection with operator lockout
  • High and low level set power alarm
  • Recipe management
  • Internal monitoring of all critical operating data
  • Diagnostic system information
  • Multiple language display

Specifications

  • Dimensions 580 x 880 x 1524-1824mm (WxDxH)
  • Power supply: single phase, 230V, 50Hz, 5amps

Internal USB data port

  • Network connectivity available

Plus

  • Stainless steel IP65 enclosure
  • Robust stainless steel fully adjustable stand for different container heights
  • Seals a variety of caps ranging from 20mm – 140mm
  • Fully compliant with all CE regulations
